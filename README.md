# Инструкция по запуску
Параметры покупаемых товаров могут передаваться через агрументы командной строки или через файл fileInput по пути src\main\java\fileInput
в формате 
```
1-3 2-4 3-4 card1
 ```
 так же организована запись в файл Print, расположенный по пути src\main\java\Print
